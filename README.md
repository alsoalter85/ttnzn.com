# ttnzn.com

`ttnzn.com` means `attenzione`.

A small static web toy built around the animated Attento mark: click the `attenzione` button to make the character shout, spawn persistent random warning text, and play a browser-generated alien-ish voice.

## Files

- `index.html` - page markup
- `styles.css` - layout, animation, and visual treatment
- `app.js` - button interaction, speech synthesis, animated mark behavior, and random text spawning
- `assets/attento.svg` - Attento SVG asset

Open `index.html` directly in a browser to run it locally.
